CSS 设置容器下子元素的间距

作者：oasis

如何设置第 2、3 个元素之间的间距为 15px 呢？

<div class="container">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
方法一：

通过给 .container 下的元素添加 class 进行控制。 

.container {
    display: flex;
}
.div-1, .div-2 {
    margin-left: 15px;
}
方法二：

通过 :first-child 进行样式覆盖。

.container {
    display: flex;
}
.container > div {
    margin-left: 15px;
}
.container > div:first-child {
    margin-left: 0;
}
方法三：

.container > div ~ div {
    margin-left: 15px;
}

方法三借助直接子元素选择器和后继选择器实现了样式控制的逻辑。
