Enums 枚举

在 TypeScript 中提供了 Enums ，它可以让程序开发人员定义一组命名常量，从而增强代码的可读性。

然而 JavaScript 中还不支持 Enums，通常可以借助常量或在工程中加入 TypeScript 来使用 Enums。除此之外也可以借助 Proxy
的特性来实现 Enums。
